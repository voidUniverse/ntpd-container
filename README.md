## ntpd container ##
A ntpd container based on CentOS 7 docker image.

## Usage ##
```
git clone https://github.com/mtt0/ntpd-container.git
cd ntpd-container
chmod +x run.sh
# Run a ntpd container
./run.sh up -d
# Stop and remove ntpd container
./run.sh down
```

## Build ##
```
git clone https://github.com/mtt0/ntpd-container.git
cd ntpd-container
chmod +x run.sh
./run.sh build
```
